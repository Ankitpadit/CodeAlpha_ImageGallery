# Image Gallery

This project is a responsive Image Gallery developed using **HTML, CSS, and JavaScript** as part of the **CodeAlpha Frontend Development Internship**.

## Features
- Responsive grid layout
- Category-based image filtering
- Lightbox view with next/previous navigation
- Hover effects and smooth transitions

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Clone or download the repository
2. Open `index.html` in your web browser
